- Standard Ubuntu 12.4 Installation from cdrom

- Installation of mysql from Sofware Center

- Tango Installation
sudo add-apt-repository ppa:abogani/tango
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys F141B61E
sudo apt-get update
sudo apt-get install tango-common tango-db tango-starter tango-test python-pytango libtango7-doc omninotify libtango7-dev libtango-java tango-starter


- Installation of g++ and javac
sudo apt-get install g++
sudo apt-get install  default-jdk

- Install ssh daemon
sudo apt-get install openssh-server
sudo restart ssh

- At this point it should be possible to run Tango locally on the machine via 
tools like "Astor"

- Then declare on the Tango database the (via Jive) the python server (neuralnet, hunveyor, etc.) you want to run and run it.
